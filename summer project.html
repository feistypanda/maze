Making the ball

then I made the ball.
The program that makes the ball bounce off of the edges of the screen uses 4 main varibles, xSpeed, ySpeed, x, and y. every frame the program adds xSpeed to x and adds ySpeed to y then draws the ball at the x and y cordinates then checks if the ball is on the left or right edge, if it is it changes xSpeed by times -1. If the ball is on the top or bottom edge it dose the same but for ySpeed. 

			var x = 200;
			var y = 200;
			var xSpeed = 1;
			var ySpeed = 1.5;
			draw = function(){
				if(page == 2){

					// add ySpeed to y
